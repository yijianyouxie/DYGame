# LINGMA - AI Assistant Created Files

本文件夹包含由 LINGMA（灵码）AI助手创建的所有脚本和文档。

## 目录结构

```
LINGMA/
├── Scripts/          # C#运行时脚本（如有）
├── Editor/           # Unity编辑器脚本（如有）
├── *.py             # Python自动化脚本
├── *.md             # Markdown文档和指南
└── *.json           # 配置文件
```

## 文件说明

### Python脚本
所有Python脚本都是用于Unity项目自动化操作的辅助工具，包括：
- UI元素创建和配置
- 场景管理
- 组件设置
- MCP工具调用等

### Markdown文档
记录各种任务的完成状态、问题解决方案和使用指南：
- 任务完成报告
- 问题修复文档
- 系统使用说明
- 快速开始指南

## 注意事项

1. **C#脚本位置**：
   - 运行时脚本应放置在 `Assets/Scripts/` 目录
   - 编辑器脚本应放置在 `Assets/Editor/` 目录
   - 本目录的 Scripts/和 Editor/子目录仅用于归档

2. **Python脚本使用**：
   - 这些脚本通常配合 MCP for Unity 工具使用
   - 需要正确的环境和配置才能运行
   - 建议先阅读相关文档再执行

3. **文件归属**：
   - 本目录仅包含AI助手创建的文件
   - 原始项目文件保持在其原有位置
   - 不会影响其他开发者创建的内容
