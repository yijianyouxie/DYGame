# 成功特效和连击系统实现说明

## 功能概述
当玩家正确点击色块后，系统会：
1. 显示华丽的成功特效（烟花效果）
2. 根据连续答对次数显示"x杀"提示
3. 特效持续1秒后自动进入下一关
4. 答错时重置连击次数并进入结果场景

## 核心修改

### GameData.cs
- 添加了 `ConsecutiveCorrect` 静态属性用于跟踪连续答对次数

### LevelController.cs
- 添加了特效相关的UI引用：
  - `successEffectPrefab`: 成功特效预制件
  - `comboText`: 连击次数显示文本
  - `effectDuration`: 特效持续时间（默认1秒）
- 修改了 `OnBlockClicked()` 方法：
  - 正确时增加连击次数并显示特效
  - 错误时重置连击次数
- 修改了 `ShowResult()` 方法：
  - 成功时直接进入下一关
  - 失败时切换到结果场景
- 新增了 `ShowSuccessEffectCoroutine()` 方法：
  - 显示烟花特效
  - 显示连击次数（当连击≥2时）
  - 带缩放动画效果

## UI设置步骤

### 1. 在LevelScene中添加UI元素

1. 打开 `LevelScene` 场景
2. 在Canvas下添加两个UI元素：
   - **成功特效**: 从 `Assets/Prefabs/SuccessEffect.prefab` 创建
   - **连击文本**: 从 `Assets/Prefabs/ComboText.prefab` 创建

### 2. 配置LevelController组件

在LevelController组件中设置以下引用：
- **Success Effect Prefab**: 拖拽成功特效预制件到此字段
- **Combo Text**: 拖拽连击文本到此字段
- **Effect Duration**: 设置特效持续时间（默认1秒）

## 特效说明

### 成功特效 (SuccessEffect.prefab)
- 简单的UI图像，可以替换为更华丽的粒子效果或动画
- 建议添加粒子系统或动画组件来增强视觉效果

### 连击文本 (ComboText.prefab)
- 显示格式："{连击次数} 杀"
- 只有在连击次数≥2时才显示
- 带有缩放动画效果

## 游戏流程

1. **答对流程**:
   - 增加连击次数
   - 显示成功特效
   - 显示连击提示（如果连击≥2）
   - 1秒后自动进入下一关

2. **答错流程**:
   - 重置连击次数为0
   - 切换到结果场景

## 后续优化建议

- **特效增强**: 将成功特效替换为真正的粒子系统或动画
- **音效**: 添加成功音效和连击音效
- **连击奖励**: 根据连击次数给予额外奖励
- **特效变体**: 不同连击次数使用不同特效
- **持久化**: 保存最高连击记录

## 技术特点

- **无缝过渡**: 成功时直接进入下一关，无需场景切换
- **状态管理**: 全局跟踪连击次数
- **动画效果**: 连击文本带有缩放动画
- **可配置**: 特效持续时间可调整
- **防作弊**: 确保特效显示完成后才进入下一关
